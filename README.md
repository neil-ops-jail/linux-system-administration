# Linux Systems Administration & Automation Portfolio
Welcome to my technincal laboratory workspace. This repository contains the product-grade scripts, security configurations, and automation workflows I deploy to manage enterprise Linux infrastructure. 
## Featured Projects & Automation

### 1. Enterprise User Onboarding Automation (`user_setup.py`)
*   **Problem:** Manual user creation across corporate infrastructure introduces human error, inconsistent folder permissions, and security vulnerabilities.
*   **Solution:** Engineered an automated, interactive Python script utilizing the `subprocess` module to create user accounts, provision dedicated home directories, enforce strict POSIX permissions, and securely assign administrative group privileges.
*   **Core Mechanics:** Python 3, System Access Control, Error Handling, Automation Engineering.
  
### 2. Permissions Management & Access Control Auditing
*   **Focus:** Enforcing the Principle of Least Privilege across multi-user environments.
*   **Implementation:** Documented system baseline configurations using `chmod`, `chown`, and administrative groups to isolate sensitive application data, secure configuration files (`/etc`), and eliminate unauthorized access vectors.
